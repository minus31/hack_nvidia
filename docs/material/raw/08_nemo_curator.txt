NeMo Curator Quick Start Guide
NeMo Curator — 대규모 데이터셋을 큐레이션하여 LLM 학습에 최적화된 고품질 데이터를 준비하는 NVIDIA의 프레임워크.
Docs: docs.nvidia.com/nemo/curator/latest/  |  GitHub: github.com/NVIDIA/NeMo-Curator
1. NeMo Curator란?

NeMo Curator는 LLM 학습을 위한 데이터 큐레이션 파이프라인을 제공합니다. 합성 데이터 파이프라인(Data Designer)으로 생성한 데이터셋의 후처리에 특히 유용합니다.

기능	설명
정확 중복 제거	해시 기반으로 완전히 동일한 문서를 제거
퍼지 중복 제거	MinHash LSH를 사용하여 의미적으로 유사한 문서를 제거
언어 식별	fastText 모델로 문서의 언어를 자동 감지 및 필터링
품질 필터링	휴리스틱 및 분류기 기반 저품질 문서 필터링
개인정보 삭제	PII(개인 식별 정보) 탐지 및 삭제/마스킹
의미 중복 제거	임베딩 기반 의미적 유사도로 중복 탐지
Track C 파이프라인에서의 NeMo Curator 위치
NeMo Data Designer (합성 데이터 생성)
        ↓
   LLM Judge 기반 필터링
        ↓
   NeMo Curator (후처리)        ← 여기!
   ├── 중복 제거
   ├── 품질 필터링
   └── PII 삭제
        ↓
   최종 학습용 데이터셋

3. Data Designer 출력에 Curator 적용하기

Data Designer로 생성한 한국어 수학 CoT 데이터셋에 NeMo Curator를 적용하여 중복을 제거하고 품질을 보장하는 예제입니다.

데이터 로드 및 정확 중복 제거
python
Copy
import nemo_curator as nc
from nemo_curator.datasets import DocumentDataset
from nemo_curator.modules import ExactDuplicates
import pandas as pd

# Data Designer 출력 로드
df = pd.read_parquet("output/korean_math_cot_full.parquet")
print(f"원본 레코드 수: {len(df)}")

# NeMo Curator DocumentDataset으로 변환
# 'problem' 컬럼을 텍스트 필드로 사용
dataset = DocumentDataset.from_pandas(df, text_field="problem")

# 정확 중복 제거: 동일한 문제를 제거
exact_dedup = ExactDuplicates(id_field="id", text_field="problem")
deduplicated = exact_dedup(dataset)
df_dedup = deduplicated.to_pandas()
print(f"정확 중복 제거 후: {len(df_dedup)}개 ({len(df) - len(df_dedup)}개 제거)")
퍼지 중복 제거 (MinHash LSH)
python
Copy
from nemo_curator.modules import FuzzyDuplicates, FuzzyDuplicatesConfig

# 퍼지 중복 제거: 유사한 문제를 탐지
fuzzy_config = FuzzyDuplicatesConfig(
    seed=42,
    char_ngrams=5,
    num_buckets=20,
    hashes_per_bucket=13,
    use_64_bit_hash=False,
    buckets_per_shuffle=5,
    false_positive_check=True,
    num_anchors=2,
    jaccard_threshold=0.8,  # 80% 이상 유사하면 중복으로 판정
)

fuzzy_dedup = FuzzyDuplicates(
    config=fuzzy_config,
    id_field="id",
    text_field="problem",
)
result = fuzzy_dedup(dataset)
df_fuzzy = result.to_pandas()
print(f"퍼지 중복 제거 후: {len(df_fuzzy)}개")
커스텀 품질 필터
python
Copy
from nemo_curator.filters import DocumentFilter

class KoreanMathQualityFilter(DocumentFilter):
    """한국어 수학 CoT 데이터셋 전용 품질 필터."""

    def __init__(self, min_problem_len=50, min_solution_len=200):
        super().__init__()
        self.min_problem_len = min_problem_len
        self.min_solution_len = min_solution_len

    def score_document(self, text: str) -> bool:
        # 최소 길이 체크
        if len(text) < self.min_problem_len:
            return False
        # 한국어 포함 여부 체크
        korean_chars = sum(1 for c in text if '\uac00' <= c <= '\ud7a3')
        if korean_chars / max(len(text), 1) < 0.3:
            return False
        return True

    def keep_document(self, score) -> bool:
        return score

# 필터 적용
quality_filter = nc.ScoreFilter(
    KoreanMathQualityFilter(min_problem_len=50),
    text_field="problem",
)
filtered_dataset = quality_filter(dataset)
print(f"품질 필터 후: {len(filtered_dataset.to_pandas())}개")
